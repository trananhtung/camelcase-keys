# Changelog

All notable changes to this project are documented here. The format is based on
[Keep a Changelog](https://keepachangelog.com/en/1.1.0/), and this project adheres to
[Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0]

### Added

- Initial release: `camelcase_keys` and `camelcase_keys_with` — a faithful port of the
  `camelcase-keys` npm package (built on the `camelcase` crate), with `deep`, `pascal_case`,
  `preserve_consecutive_uppercase`, `exclude`, and `stop_paths` options.
